# Trade & Investment Management Portal (TIMP)

Afreximbank is developing the Trade & Investment Management Portal (TIMP) as a centralized digital ecosystem to improve end-to-end event execution (roadshows, forums) and convert engagement into measurable deal origination and capital mobilisation outcomes.

This repository contains the product handover pack (PRD, workflow, data dictionary, RBAC) plus sample seed data to accelerate UI build and demos.

## MVP scope (for initial build + demo)
**MVP = Registration + Deal Room + CRM sync (stubbed) + basic analytics.**

### In scope
- Event calendar and event pages (public view)
- Registration + verification workflow (admin + public)
- Deal Room (browse/filter + deal detail)
- “Request for Prospectus” and 1:1 meeting scheduling (lightweight)
- Digital Deal Book generation (exportable dataset + placeholder PDF generation)
- Basic analytics dashboard (attendance, engagement, downloads)

### Out of scope (Phase 2+)
- Virtual Exhibition
- Advanced AI matchmaking
- Full Africa Trade Gateway (ATG) ecosystem integration
- Full knowledge repository / audit search UI (can be designed but not fully built in MVP)

## Repo structure
- `/docs` – product documents developers build against
- `/sample-data` – JSON seed data for local dev + demo
- `/design` – exported screens / links (optional)
