# RBAC — Roles & Permissions (MVP)

## Roles
### Admin (internal)
Assigned to specific internal functions (COEV/TFIP/CLRE/Regional Staff/Digital Business).

### Public (verified external)
Registered and verified participants (investors, project owners, etc.).

## Permissions matrix (MVP)
| Capability | Admin | Verified user |
|---|---:|---:|
| View public event calendar/pages | ✅ | ✅ |
| Create/edit/publish events | ✅ | ❌ |
| Upload programme/materials/recordings | ✅ | ❌ |
| Approve/reject registrations | ✅ | ❌ |
| View Deal Room (published deals) | ✅ | ✅ |
| Create/edit/publish opportunities | ✅ | ❌ |
| Access verified-only materials (secure data room) | ✅ | ✅ |
| Access admin-only materials | ✅ | ❌ |
| Request prospectus | ✅ | ✅ |
| Schedule 1:1 meetings | ✅ | ✅ |
| View analytics dashboards | ✅ | ❌ (optional limited view) |
| Export analytics | ✅ | ❌ |
| Manage users / suspend access | ✅ | ❌ |
