# Data Dictionary — TIMP (MVP)

## 1) Event
- id (uuid)
- title (string)
- region (enum: north/east/west/southern/central/caribbean)
- country (string)
- city (string)
- venue (string)
- start_date (date)
- end_date (date)
- theme (string)
- banner_url (string)
- programme_files (array of Material refs)
- status (enum: draft/published/live/post_event/archived)

## 2) User
- id (uuid)
- full_name (string)
- email (string)
- organisation_id (uuid)
- title (string)
- role (enum: admin/public_verified)
- verification_status (enum: registered/verified/suspended)
- preferred_language (enum: en/fr/ar/pt)
- region_affiliation (optional)

## 3) Organisation
- id (uuid)
- name (string)
- type (enum: investor/project_owner/partner/tpo/exporter/other)
- country (string)
- sector (string)
- website (string)

## 4) Registration
- id (uuid)
- event_id (uuid)
- user_id (uuid)
- status (enum: registered/verified/rejected)
- created_at (datetime)
- verified_by (user_id, nullable)
- verified_at (datetime, nullable)

## 5) Opportunity (Deal Room)
- id (uuid)
- title (string)
- country (string)
- sector (string)
- ticket_size_usd (number)
- summary (text)
- sponsor_org_id (uuid)
- status (enum: draft/published/engaged/follow_up/closed_won/closed_lost)
- pitch_video_url (string, nullable)
- attachments (array of Material refs)
- created_at (datetime)
- updated_at (datetime)

## 6) Material
- id (uuid)
- type (enum: programme/deck/recording/deal_book/prospectus/other)
- title (string)
- storage_url (string)
- access_level (enum: public/verified_only/admin_only)
- uploaded_by (user_id)
- uploaded_at (datetime)

## 7) ProspectusRequest
- id (uuid)
- opportunity_id (uuid)
- user_id (uuid)
- requested_at (datetime)
- status (enum: submitted/fulfilled/declined)
- notes (text, nullable)

## 8) Meeting
- id (uuid)
- event_id (uuid)
- opportunity_id (uuid, nullable)
- requester_user_id (uuid)
- requested_with_org_id (uuid)
- modality (enum: physical/virtual)
- scheduled_start (datetime)
- scheduled_end (datetime)
- location_or_link (string)
- status (enum: requested/confirmed/completed/cancelled)

## 9) Feedback
- id (uuid)
- event_id (uuid)
- user_id (uuid)
- nps (integer 0-10, nullable)
- rating (integer 1-5, nullable)
- comments (text, nullable)
- submitted_at (datetime)

## 10) AnalyticsEvent (event log)
- id (uuid)
- event_id (uuid, nullable)
- user_id (uuid, nullable)
- opportunity_id (uuid, nullable)
- type (enum: page_view/download/prospectus_request/meeting_scheduled/survey_submitted/etc.)
- metadata (json)
- occurred_at (datetime)
